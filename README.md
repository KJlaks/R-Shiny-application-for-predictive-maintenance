Inspired by the Siemens predictive maintenance introduction in their company, I started this project while doing my course in Data analytics and Ai.
The idea of this application is to simulate the data using generative AI and use that dataset to predict the future failures of the machines.
Before the failure occurs the workers will be able to get an idea on which machine requires maintanence even befor ethe failure actually occurs.
It is possible by using different parameters that is related to the health of the machines.
